# SkyStore Pro: E-commerce & Content Management System

![Python](https://img.shields.io/badge/python-3.11+-blue.svg)
![Django](https://img.shields.io/badge/django-5.x-green.svg)
![Redis](https://img.shields.io/badge/redis-caching-red.svg)

**SkyStore Pro** — это комплексное веб-приложение для электронной коммерции, объединяющее в себе функционал онлайн-магазина и контентной платформы (блога). Проект демонстрирует навыки работы с классической MTV-архитектурой Django.

## Основной функционал
* **Каталог товаров:** Гибкая система управления продуктами, категориями и версиями товаров.
* **Блог-платформа:** Полноценный модуль для публикации статей с поддержкой динамических URL и счетчиком просмотров.
* **Личный кабинет:** Система регистрации и авторизации пользователей с восстановлением пароля.
* **Кэширование:** Интеграция с Redis для оптимизации загрузки страниц категорий и продуктов.
* **Контекстные формы:** Использование Django Forms для управления контентом непосредственно из интерфейса сайта.

## Технологический стек
* **Framework:** Django (MTV architecture).
* **Database:** PostgreSQL.
* **Caching:** Redis.
* **Frontend:** Django Templates, Bootstrap 5.
* **Environment:** Poetry (управление зависимостями).

## Архитектура проекта
Проект разделен на логические приложения:
* `catalog/`: Логика товаров, категорий и управления версиями.
* `blog/`: Система управления контентом и новостями.
* `users/`: Модуль аутентификации и профилей.

## Установка и запуск

1. Клонируйте репозиторий:

    git clone https://github.com/AJLbN0H/skystore-pro-engine.git

2. Настройте окружение:
Создайте файл .env и заполните его по примеру .env.example (настройки БД, SECRET_KEY, настройки Redis).

3. Установите зависимости:

    poetry install

4. Запустите проект:

    python manage.py migrate
    python manage.py runserver
