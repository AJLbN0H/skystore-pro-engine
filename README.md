# SkyStore Pro: Масштабируемая Backend-платформа на Django

SkyStore Pro — это современный веб-сервис для управления товарным каталогом и контентом, спроектированный с упором на производительность и безопасность. Проект реализует полноценный цикл управления данными: от гибкой настройки моделей пользователей до оптимизации запросов к БД через механизмы кэширования.

## Технологический стек и архитектура

Проект построен на базе Django Framework с использованием надежных промышленных инструментов:

* Backend: Python 3.11+, Django 4.2+ (MVC-архитектура).
* База данных: PostgreSQL для надежного хранения транзакционных данных.
* Кэширование: Redis для ускорения отклика системы и снижения нагрузки на БД.
* Управление зависимостями: Poetry (поддержка pyproject.toml).
* Работа с изображениями: Pillow.

## Ключевые архитектурные решения

### 1. Кастомная система аутентификации
Реализована собственная модель пользователя (AbstractUser), где в качестве основного идентификатора (USERNAME_FIELD) используется Email. Это современный стандарт для e-commerce платформ, повышающий удобство для клиентов.

### 2. Разграничение прав (RBAC)
В системе реализована матрица доступа:
* Владельцы (Owners): Обладают полным контролем над своими объектами (товарами/записями).
* Модераторы: Имеют расширенные полномочия, такие как управление признаком публикации (permission: can_unpublish_product), но ограничены в деструктивных действиях.

### 3. Оптимизация производительности (Caching)
Для обеспечения быстродействия внедрено кэширование:
* Per-view caching: Кэширование страниц с детальной информацией о товарах.
* Low-level API: Кэширование выборок QuerySet в контроллерах для исключения повторных тяжелых запросов к PostgreSQL.

### 4. Паттерн Service Layer
Для разгрузки контроллеров (views) и соблюдения принципа единственной ответственности, бизнес-логика вынесена в отдельный слой сервисов (services.py).

## Инструкция по развертыванию

Для запуска проекта необходимо наличие установленного PostgreSQL и запущенного Redis-сервера.

1. Клонирование репозитория:
   git clone https://github.com/ваш-логин/skystore-pro-engine.git

2. Настройка окружения:
   Создайте файл .env в корне проекта и заполните его по примеру:
   SECRET_KEY=ваш_секретный_ключ
   DEBUG=True (или False для продакшена)
   NAME=db_name
   USER=db_user
   PASSWORD=db_password
   HOST=127.0.0.1
   PORT=5432

3. Установка зависимостей:
   poetry install
   Или через pip:
   pip install -r requirements.txt

4. Миграции и база данных:
   python manage.py migrate

5. Наполнение данными:
   Для первичного наполнения каталога выполните:
   python manage.py loaddata catalog_fixture.json

6. Запуск:
   python manage.py runserver
