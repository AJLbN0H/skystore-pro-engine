# SkyStore Pro: Масштабируемая Backend-платформа на Django

SkyStore Pro — это современный движок для управления товарным каталогом и контентом, спроектированный с упором на производительность и безопасность. Проект реализует полноценный цикл управления данными: от кастомной авторизации пользователей до оптимизации запросов к БД через механизмы кэширования.

## Технологический стек и архитектура

Проект построен на базе Django Framework с использованием надежных инструментов:

* Backend: Python 3.13+.
* Framework: Django 4.2+.
* База данных: PostgreSQL (через psycopg2).
* Кэширование: Redis (интеграция через django.core.cache).
* Управление зависимостями: Poetry (стандарт pyproject.toml).
* Работа с изображениями: Pillow.

## Ключевые архитектурные решения

### 1. Кастомная система аутентификации
В проекте реализована модель пользователя на базе AbstractUser, где основным идентификатором (USERNAME_FIELD) является Email. Это упрощает взаимодействие с платформой и соответствует современным стандартам e-commerce.

### 2. Разграничение прав доступа (RBAC)
Реализована гибкая матрица прав:
* Владельцы (Owners): Обладают полным контролем над своими товарами.
* Модераторы: Имеют право на управление публикацией объектов (can_unpublish_product), что позволяет контролировать контент без доступа к полному удалению.

### 3. Оптимизация производительности (Caching)
Для обеспечения быстродействия под нагрузкой внедрено:
* Per-view caching: Кэширование страниц с детальной информацией о товарах (60*15 сек).
* Low-level API: Кэширование выборок QuerySet в контроллерах через Redis для исключения повторных запросов к PostgreSQL.

### 4. Интеграция с почтовыми сервисами
Настроена работа с SMTP-сервером (Yandex) для обеспечения автоматической рассылки уведомлений и подтверждений.

### 5. Паттерн Service Layer
Бизнес-логика (например, фильтрация продуктов по категориям) вынесена в слой сервисов (services.py), что позволяет сохранять контроллеры «тонкими» и упрощает тестирование.

## Инструкция по развертыванию

Для запуска проекта необходимо наличие PostgreSQL и запущенного Redis-сервера.

1. Клонирование репозитория:
   ```git clone https://github.com/AJLbN0H/skystore-pro-engine.git```

2. Настройка окружения:
   Создайте файл .env в корневой директории и заполните его своими данными. 

4. Установка зависимостей:
   ```poetry install```
   Или:
   ```pip install -r requirements.txt```

5. Миграции и база данных:
   ```python manage.py migrate```

6. Наполнение данными:
   Загрузите демонстрационный каталог:
   ```python manage.py loaddata catalog_fixture.json```

7. Запуск:
   ```python manage.py runserver```
